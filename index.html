Fleetwood Mac's website

Het eerste project dat ik voor mijn opleiding heb moeten doen en tevens de eerste website die ik ooit heb gebouwd. Het doel was om grip te krijgen op HTML en CSS en betreft een simpele, informatieve website over de band Fleetwood Mac. Exhibit 2:

Custom Artist

Het derde project dat ik voor mijn opleiding heb gemaakt. Het doel was om het gebruik van databases te implementeren in websites. Custom Artist betreft een vraag- en -aanbod platform voor kunstenaars. Het biedt kunstenaars de mogelijkheid om in contact te komen met cliënten. Deze cliënten kunnen zowel bedrijven als particulieren zijn die op hun beurt op zoek zijn naar kunstenaars om gericht een kunstwerk te maken.

Exhibit 3:

Barbie Booth

Elk mens heeft een verzameling. Mijn moeder heeft een fantastisch grote Barbie-verzameling tussen de jaren ’60 en ’80. Als kind kan ik niet genoeg uitdrukken hoe trots ik daar op ben. Gelukkig is ze dat zelf ook, maar stel nou dat ze (ooit) haar poppen wil verkopen? Via Marktplaats gaat het goed, maar liever via Barbie Booth. Dit was mijn afstudeerproject bij Code Institute en ik blijf er trots op.
